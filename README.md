# Simon Music Release

This repository stores public update metadata and release assets for Simon Music.

## Metadata

- Desktop: `publish/desktop/version.json`
- Mobile: `publish/mobile/version.json`

## GitHub Release Assets

Use tags such as `v5.0.1`.

Required desktop assets:

- `desktop-latest-x64.yml`
- `desktop-latest-ia32.yml`
- `simon-music-desktop-v5.0.1-x64-Setup.exe`
- `simon-music-desktop-v5.0.1-x64-Setup.exe.blockmap`
- `simon-music-desktop-v5.0.1-ia32-Setup.exe`
- `simon-music-desktop-v5.0.1-ia32-Setup.exe.blockmap`

Required mobile assets:

- `simon-music-mobile-v5.0.1-arm64-v8a.apk`
- `simon-music-mobile-v5.0.1-armeabi-v7a.apk`
